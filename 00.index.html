    • 클래스 선택자는 . 점을 찍어서 표시해서 사용한다 

        선택자[속성=속성값] { 스타일 속성: 값} 특정 값을 단어로 포함하는 태그선택 띄어쓰기로 구분
    선택자[속성~=속성값] {스타일 속성: 값} 특정 값을 단어로 포함하는 태그 선택 "-"으로 구분 -앞의  내용이 동일
    선택자[속성!=속성값] {스타일 속성: 값} 
    선택자[속성^=속성값] {스타일 속성: 값}
    선택자[속성$=속성값] {스타일 속성: 값}
    선택자[속성*=속성값] {스타일 속성: 값}
